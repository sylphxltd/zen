# ZenJS Official Website Plan

## Architecture
```
website/
├── src/
│   ├── pages/
│   │   ├── Home.tsx          # Landing page
│   │   ├── Docs.tsx          # Documentation
│   │   ├── Examples.tsx      # Interactive examples
│   │   ├── Benchmarks.tsx    # Performance comparisons
│   │   └── Playground.tsx    # Live code editor
│   ├── components/
│   │   ├── Header.tsx
│   │   ├── Footer.tsx
│   │   ├── CodeBlock.tsx
│   │   └── Demo.tsx
│   ├── App.tsx              # Root component
│   └── main.tsx             # Entry point
├── public/
│   └── style.css
└── index.html
```

### 3. Context API

### 5. Suspense/Resource
